# Golang test TeamCity converter

Convert go test output to TeamCity format

Support Run, Skip, Pass, Fail

### Installation

```bash
go get github.com/2tvenom/go-test-teamcity
```

### How use
```bash
go test -v ./... | go-test-teamcity
```
